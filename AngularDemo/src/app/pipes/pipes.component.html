PIPES IN ANGULAR

PIPES Used to Usage
uppercase converts the template expression into uppercase. {{ "Laptop" | uppercase }}
lowercase converts the template expression into lowercase. {{ "LAPTOP" | lowercase }}
titlecase converts the first alphabet in each word of the given expression into an uppercase alphabet. {{ "product details" | titlecase }}
currency displays the currency symbol before the expression. Default- currency symbol $ {{ 66600 | currency }}
{{ 66600 | currency : 'CAD' }}
{{ 66600 | currency :'INR' }}
{{ 66600 | currency :'INR' :'code' }}
{{ 66600 | currency :'INR' :'symbol' }}
{{ 66600 | currency :'INR' :'myOwnString' }}
{{ 66600 | currency :'INR' :'code' :'6.2-3'}}
{{ 66600 | currency :'INR' :'code' :'1.3'}}
percent can be used to display the number as a percentage. {{ 0.1 | percent }} will display 10%
{{ 0.1 | percent:'2.2-3' }}
number can be used to format a number {{ 25000 | number }} {{ 25000 | number:'.3-5' }}
date can be used to display the date in the required format. {{ "6/2/2017" | date }}
{{ "6/2/2017, 11:30:45 AM" | date:'medium' }}
{{ "6/2/2017, 11:30:45 AM" | date:'mmss' }}
slice can be used to extract a subset of elements or characters from an array or string respectively. {{ ['a','b','c','d']| slice:2}}
{{ ['a','b','c','d']| slice:1:3}}
{{ 'Laptop Charger'| slice:3:6}}
{{ 'Laptop Charger'| slice:-4}}
{{ 'Laptop Charger'| slice:-4:-2}}
json used to displays the given expression in the form of a JSON string.used for debugging. {{ {'productId':1234, 'productName':'Samsung Mobile'} | json }}
Chaining of pipes Multiple pipes can be chained together to get a more desirable output. The final output will be displayed with all the pipes applied.

Month is {{ "2018-11-11" | date | uppercase | slice:0:4 }}
